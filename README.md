# LISTA DE PRESENTES DE CASAMENTO

## Descrição

Quando decidimos nos casar, resolvemos realizar uma cerimônia em outro estado, por isso não teríamos como levar os presentes para nossa casa.
Então tivemos a ideia de criar uma lista de presentes de casamento colaborativa. 

A ideia é que os noivos possam criar uma lista de presentes e compartilhar com os convidados, que poderão escolher um presente para dar comprando diretamente da loja sugerida. Assim, os noivos não correm o risco de ganhar presentes repetidos ou que não gostem. Ou até mesmo, os convidados podem optar por dar um valor em dinheiro, via Pix.


## Tecnologias

- React
- Typescript

## Scripts Disponíveis

No diretório do projeto, você pode executar:

Executa o aplicativo no modo de desenvolvimento.\

A página será recarregada se você fizer edições.\
Você também verá quaisquer erros de lint no console.

### `npm test`

Inicia o executor de testes no modo de observação interativa.\
Veja a seção sobre [execução de testes](https://facebook.github.io/create-react-app/docs/running-tests) para mais informações.

### `npm run build`

Compila o aplicativo para produção na pasta `build`.\

A compilação é minificada e os nomes dos arquivos incluem os hashes.\
